# Implementation Plan

## 方針

- 展示モードは、既存のゲーム体験や判定ロジックを変えず、運用上の復帰とリセットだけを追加する。
- まず自動復帰対象や待機時間を `src/game/` の純粋ロジックに分離し、ReactやブラウザAPIへ依存させない。
- 自動復帰と手動リセットは、既存の `resetGameSession("title")` と `releaseMediaResources()` に統一する。
- プレイ中のリセットUIは控えめに配置し、HUD、壁、安全領域、判定フィードバックを妨げない。
- E2Eで長時間の自動復帰待機は避け、ユニットテストと手動確認を中心にする。
