# Product Overview

`steer-clear-of-the-wall` は、カメラの前で体を動かし、迫ってくる壁を避ける
ブラウザ向け体験型ゲームです。プレイヤーの姿勢を検出し、画面上のアバターに反映して、
壁パターンの穴に体の形を合わせる遊びを提供します。

このプロジェクトは初期段階です。2026-06-24時点でREADMEはプロジェクト名のみで、
`package.json` と `src/` は未作成です。以後の仕様や実装は、本ステアリングを
出発点として独自に設計します。

## Core Experience

ゲーム体験の中心は次の流れです。

1. Camera Input
2. Pose Detection
3. Avatar Display
4. Wall Pattern
5. Collision Judgment
6. Score / Result

カメラ入力は操作手段であり、最終的にユーザーが見る主役はゲーム画面です。
検出した姿勢をアバターとして表示し、迫る壁の形に合わせて体を動かす体験を重視します。

## Core Capabilities

- カメラからプレイヤーの姿勢を取得する。
- 姿勢検出結果を、直接のカメラ映像ではなくアバター表示へ変換する。
- 複数の壁パターンを出現させ、プレイヤーの姿勢と照合する。
- 成功、失敗、スコア、結果を短時間で分かりやすく提示する。
- カメラ権限や姿勢検出失敗時に、ユーザーがリトライできる状態を用意する。

## Target Use Cases

- ブラウザで手軽に試せる体験型ゲーム。
- 展示、授業、イベントなど、短時間で理解して遊べるデモ。
- 姿勢検出、Canvas描画、ゲームロジックを組み合わせた学習用プロジェクト。

## Value Proposition

体をコントローラーにすることで、キーボードやゲームパッドなしに直感的に遊べます。
また、カメラ映像をそのまま表示するのではなくアバター化することで、ゲームらしい
見た目とプライバシーへの配慮を両立します。
