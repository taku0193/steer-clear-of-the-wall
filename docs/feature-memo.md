# Feature Memo

## 1. 作りたいもの

体を使って迫る壁を避ける体験型ゲームを作る。

## 2. 参考にするもの

参考リポジトリ:

https://github.com/taku0193/ai-wall

ただし、コードはコピーしない。
参考にするのはシステムの流れだけ。

参考にする流れ:

- カメラ入力
- 骨格検出
- アバター表示
- 迫る壁の表示
- 壁との当たり判定
- スコア表示
- 結果表示

## 3. 目的

オープンキャンパスなどで、高校生や来場者が短時間で体験できる分かりやすいゲームを作る。

カメラの前で体を動かすことで、AIや骨格推定の仕組みを直感的に体験できるようにする。

## 4. 想定ユーザー

- 高校生
- オープンキャンパスの来場者
- 研究室展示を見る人
- AIや画像認識を初めて体験する人

## 5. 基本のゲームの流れ

1. タイトル画面を表示する
2. 開始ボタンを押す
3. カメラ使用を許可する
4. 骨格検出を開始する
5. 画面上にアバターを表示する
6. カウントダウンを表示する
7. 奥から壁が迫ってくる
8. プレイヤーは体を動かして壁の穴に合わせる
9. 成功・失敗を判定する
10. スコアを表示する
11. 制限時間が終わったら結果画面を表示する

## 6. 今回作る最小機能

まずは最小構成で作る。

- タイトル画面
- カメラ開始ボタン
- 骨格検出
- 簡単なアバター表示
- 1種類または数種類の壁パターン
- 簡易的な当たり判定
- スコア表示
- 結果画面
